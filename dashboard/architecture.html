Event-Driven Trigger Webhook

Instead of relying on slow polling intervals, DataTrust registers an OmEventSubscriber. When metadata changes in OpenMetadata (like adding an owner), OM instantly pings the engine to trigger a targeted rescore.

Deep Metadata Fetch API

The engine securely connects back to OpenMetadata using Bot Tokens to fetch dependencies. It parses the Lineage API for graph traversal, and the native Profiler API to extract granular row counts and null ratios.

Score Writeback JSON Patch

Trust scores aren't trapped in an external dashboard. The engine modifies the OM Schema using the Custom Properties API, and writes the `trustScore` and `trustGrade` directly into the OpenMetadata table entity using JSON Patch.

Incident Response Auto-Task

If a table's score drops below the Critical Threshold (e.g., 50/100), the engine acts. It automatically sends a webhook to Slack, and uses the OM Feed API to instantly create a Data Quality Task assigned to the table owner.
